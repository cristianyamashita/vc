      How it works The board chooses a fresh subset from the language-specific word array each round. A different placement strategy changes the order, anchor direction, and crossing bias. For Japanese, the answers are kana only: hiragana or katakana.
      Controls Type one character per cell. Arrow keys move around the board. Click a clue to highlight its word, then keep typing to move forward automatically. Hint reveals one character in the selected clue.
      Single-file HTML game with local language and theme preferences.
